# 02 WBS 文档

> 要求：WBS 必须围绕交付物（成果）分解，至少分解到三级。

## 1. 一级交付物
1.管理端交付物：面向管理人员的活动组织与数据处理成果  
2.客户端交付物：面向普通学生的报名与签到交互成果   
3.项目管理成果：确保项目有序进行的各类过程文档与协作留痕

## 2. 二级与三级分解
示例格式：

1. 项目管理成果
   1.1 启动成果
   1.1.1 仓库与目录结构
   1.1.2 团队角色分工表
   1.2 计划成果
   1.2.1 WBS 文档
   1.2.2 简化进度计划

2. 业务功能成果
   2.1 活动管理成果
   2.1.1 活动信息结构说明
   2.1.2 活动发布页面草案

## 3. 分解说明
请说明本组的分解思路，特别是为什么这样划分交付物。
