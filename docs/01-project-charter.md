# 01 项目启动说明

## 1. 项目名称
校园活动报名与签到管理系统

## 2. 项目目标
-	支持活动管理人员发布活动信息并维护报名时间、人数上限和活动地点。
-	支持学生查看活动并完成在线报名。   
-	支持活动组织者对报名名单进行审核与查看。   
-	支持现场签到，并形成签到结果统计。   
-	支持活动结束后导出基本统计结果。

## 4. 项目边界
（1）纳入范围

本项目在实验范围内主要包括以下内容：

活动信息发布与维护
学生报名信息提交
报名名单查看与审核
签到记录与签到状态管理
签到统计结果导出
项目管理相关文档的编制（WBS、进度计划、配置管理等）
基于 Git 的版本管理与团队协作
（2）不纳入范围

为保证实验重点突出，本项目不包括以下内容：

复杂支付或收费功能
与真实校园统一身份认证系统的对接
高并发处理与系统性能优化
完整业务系统开发与部署上线
移动端 App 或小程序的正式开发
